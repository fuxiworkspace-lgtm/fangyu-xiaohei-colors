# 房语胶囊款小黑固定形体

所有彩色小黑共享同一个胶囊款角色形体。颜色只能改变身体主色，不允许改变轮廓、比例、眼睛、四肢、表情和动作逻辑。

## 固定识别点

- 小黑是一个黑色或彩色实心胶囊小人，像低调的桌面操作员，不是吉祥物，也不是表情包角色。
- 身体是竖向胶囊 / 药丸形的一体化圆角轮廓，上下圆润，中段略直。
- 整体比例约为高宽 `1.8:1` 到 `2.4:1`，可以轻微胖瘦变化，但不能变成圆球、豆子、分离头身或大头小身。
- 角色没有单独头部和躯干分区；整个人就是一个连续胶囊体。
- 眼睛是两个小而清楚的白色椭圆或白点，位置靠近，通常不需要黑色瞳孔；表情空、呆、冷静、认真。
- 没有嘴，或只有极简短线嘴；不要笑脸、夸张表情、眉毛和复杂五官。
- 四肢极细但清楚可见，像手绘出来的小胳膊小腿；手掌可以抓、推、拉、拧、搬真实物件。
- 轮廓略微不规则，有手绘感和轻微实物玩偶质感；可以有很轻的哑光颗粒，但不要变成光滑塑料 3D 吉祥物。
- 不穿复杂衣服，不戴帽子，不靠服饰解释身份。
- 尺寸通常是画面高度的 8%-13%，在长卷里可以更小，但每个节点都要清楚可见。

## 动作原则

- 必须和真实物件发生物理关系：推、拉、拧、抬、绑、修、贴、夹、扛、钻入、拖出。
- 不能只是站在旁边看，也不能只是举牌说明主题。
- 可爱来自“认真做荒诞事”，不是卖萌。

## 颜色变体规则

彩色版本只替换身体颜色。必须同时保留：

- 竖向胶囊身体；
- 一体化圆角轮廓；
- 没有头身分离；
- 白色小椭圆眼或白色小点眼；
- 极细四肢；
- 冷静认真表情；
- 哑光手绘 / 轻实物质感；
- 认真做事的物理动作逻辑。

禁止：

- 因颜色变化而改成不同物种、宠物、吉祥物、超级英雄或职业装角色。
- 用帽子、衣服、发型、眉毛、笑脸、饰品区分身份。
- 变成纯色剪影而看不清白色眼睛、四肢和动作。
- 变成大圆脑袋 + 小身体的分体角色。
- 变成圆滚滚球形宠物、豆子、Minions 风格角色或高光塑料玩具。
